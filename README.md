# GIA 逆向工作区

本目录用于保存 `.gia` 文件逆向分析资料、样本文件、生成结果和自动化脚本。

| 文件或目录 | 作用 |
| --- | --- |
| `scripts/` | 所有可执行脚本，按用途拆分为镜头、元件变量解析、元件变量导入三类。 |
| `scripts/gil_workflow/` | 从地图 `.gil` 提取结构体，并按规格生成多个新元件和变量的工作流脚本。 |
| `外部项目萃取/` | 对外部千星逆向项目的字段语义、生成逻辑和可复用结论的本地整理。 |
| `元件变量自动化/` | 元件变量相关样本、导入结果、合并结果和解析 JSON/CSV。 |
| `结构体导入导出分析/` | 结构体导入/导出行为分析样本。 |
| `.history/` | 编辑器或工具生成的历史记录目录。 |
| `镜头.gia` | 镜头类基础样本。 |
| `0-180raid.gia` | 镜头角度范围验证样本。 |
| `滑动变焦镜头组.gia` | 滑动变焦镜头组样本。 |
| `镜头组.json` | 镜头组配置样本。 |
| `高密度镜头模板_FOV30-60_距离1-7.csv` | 高密度镜头参数模板表。 |
| `gia_guess.proto` | 当前推测的 `.gia` Protobuf schema。 |
| `GIA解析文档.md` | 主要中文解析文档。 |
| `GIL解析文档.md` | 千星奇遇 `.gil` 完整工程存档初步解析文档。 |
| `GIA_FORMAT_NOTES.md` | 英文格式笔记。 |
| `NUL` | 历史遗留文件，当前脚本不依赖。 |
| `README.md` | 当前目录内容说明。 |
